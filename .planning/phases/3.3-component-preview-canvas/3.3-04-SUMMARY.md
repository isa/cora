### Auto-fixed Issues

**1. [Rule 3 - Blocking] Isolated npm cache for `npm pack` in smoke**
- **Found during:** Clean-install smoke verification.
- **Issue:** `npm pack` used the user's global npm cache before the script set an isolated cache for install, causing permission failures in this environment.
- **Fix:** Set `npm_config_cache="$SMOKE_DIR/npm-cache"` for `npm pack` as well as install.
- **Files modified:** `packages/cora/tests/smoke/clean-install.sh`
- **Verification:** `bash packages/cora/tests/smoke/clean-install.sh` passed outside sandbox.

**Total deviations:** 1 auto-fixed. **Impact:** Improves smoke hermeticity without changing package behavior.
