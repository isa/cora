<objective>
Define the internal preview pack, typed control schema, scenario model, and two-node workbench state that the UI and canvas will consume.
</objective>

<threat_model>
| Threat | Severity | Mitigation |
|--------|----------|------------|
| T-3.3-04: Control schema allows invalid props and crashes renderer components | medium | Use discriminated constrained control definitions and test enum, color, number, boolean, text, and size defaults. |
| T-3.3-05: Pack manifest becomes accidental public API too early | low | Keep manifest types internal to `src/preview` and do not add a `cora/preview` package export in Phase 3.3. |
| T-3.3-06: External pack loading leaks into this phase | medium | Built-in pack is static and internal; no filesystem/network extension discovery is added. |
</threat_model>
