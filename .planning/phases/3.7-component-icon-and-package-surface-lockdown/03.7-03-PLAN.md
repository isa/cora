The test should:

1. Run `bun pm pack` (or `npm pack --pack-destination <tmpdir>`) to create the tarball in a temp directory inside the workspace (e.g., `packages/cora/tests/smoke/.tmp/`).

2. Extract the tarball (using `tar xzf` via child_process).

3. Collect the list of all files in the extracted package directory.

4. Assert REQUIRED files exist:
   - `package/dist/cli.js` (CLI entry point)
   - `package/dist/index.js` (main library)
   - `package/dist/core/index.js` (core subpath)
   - `package/dist/renderer/components/index.js` (components subpath)
   - `package/dist/renderer/assets/fonts/NotoSans-Regular.ttf` (bundled font)
   - `package/dist/renderer/assets/fonts/NotoSans-SemiBold.ttf` (bundled font)
   - `package/SKILL.md`
   - `package/package.json`

5. Assert FORBIDDEN patterns do NOT exist:
   - Any file matching `**/preview/**` or `**/preview-dist/**`
   - Any file matching `**/*.ts` (except `.d.ts` files)
   - Any file matching `**/vite.config*` or `**/tsdown.config*`
   - Any file matching `**/src/**`
   - Any file matching `**/.planning/**`
   - Any file matching `**/node_modules/**`

6. Assert total tarball size is reasonable (< 5MB uncompressed — current useful content is ~2MB of fonts + ~500KB of JS).

7. Clean up the temp directory in afterAll.

Use `execSync` from `child_process` for shell commands. Use `readdirSync` recursive or a glob walk for file listing.
