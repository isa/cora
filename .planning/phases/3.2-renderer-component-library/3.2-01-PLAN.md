  <action>
Create `packages/cora/src/renderer/components/lines/`.

In `lines/styles.ts`, define:

- `export type LineStyle = 'solid' | 'dashed' | 'dotted'`
- `lineDasharray(lineStyle: LineStyle | undefined, strokeWidth: number | undefined): string | undefined`

Use `solid` as the default. Use the same dash math as `borderDasharray`, with default stroke width `1`.

In `lines/markers.tsx`, define:

- `export type MarkerType = 'none' | 'arrow' | 'circle' | 'filledCircle'`
- stable marker ids: `cora-marker-arrow`, `cora-marker-circle`, `cora-marker-filled-circle`
- `markerUrl(marker: MarkerType | undefined): string | undefined`, returning `undefined` for `none` or `undefined`
- `LineMarkerDefs({ color = 'currentColor' }: { color?: string })`, rendering `<marker>` definitions for arrow, circle, and filledCircle

In `lines/Line.tsx`, define:

- `export interface LineProps` with `points: Array<{ x: number; y: number }>; lineStyle?: LineStyle; strokeColor?: string; strokeWidth?: number; startMarker?: MarkerType; endMarker?: MarkerType`
- `export function Line(props: LineProps)`

`Line` must render a single `<path>` from the provided points. It must not run layout, mutate points, read DOM APIs, parse YAML, or compute endpoints from node ids. Default `strokeColor` to `currentColor`, `strokeWidth` to `1`, and `lineStyle` to `solid`.

Export `Line`, `LineProps`, `LineStyle`, `MarkerType`, `LineMarkerDefs`, `markerUrl`, and `lineDasharray` from `components/index.ts`.
  </action>
