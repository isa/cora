## Recommended Schema Shape

Add optional `diagram.grid` to `diagramBase`:

```yaml
diagram:
  grid:
    spacing: 16        # default 16 when omitted
    majorEvery: 5      # default 5 when omitted
    visible: true      # edit-surface only; ignored by SVG/PDF/PNG renderers
```

- All fields optional; resolver applies defaults (D-01, D-04).
- `visible` affects preview/canvas only (D-09).
- Do **not** add `snapLayout` unless trivial ŌĆö defer ELK post-snap (D-10).

## Core Module API (recommended)

File: `packages/cora/src/core/grid.ts`

```typescript
export const DEFAULT_GRID_SPACING = 16;
export const DEFAULT_GRID_MAJOR_EVERY = 5;
export const DEFAULT_GRID_ORIGIN = { x: 0, y: 0 } as const;

export interface GridConfig {
  spacing: number;
  majorEvery: number;
  visible: boolean;
}

export function resolveGridConfig(grid?: Partial<GridConfig>): GridConfig;
export function snapScalar(value: number, spacing: number, origin?: number): number;
export function snapPoint(point: { x: number; y: number }, config?: GridConfig): { x: number; y: number };
export function snapSize(size: { width: number; height: number }, config?: GridConfig): { width: number; height: number };
```

Export from `core/index.ts` for `cora/core` subpath (D-15).

**Snap math:** `snapScalar(v, spacing, origin=0) => origin + Math.round((v - origin) / spacing) * spacing`

## Preview Integration Points

1. **Visual grid:** Update CSS to 16px minor dots OR add SVG `<pattern>` in diagram coords aligned to viewBox (world-aligned survives pan/zoom better). Recommend SVG pattern in `WorkbenchCanvas` defs using `spacing` from resolved config ŌĆö CSS alone drifts under pan.

2. **Snap gate:** `shouldSnap = snapEnabled && !shiftKeyDown` (D-05).

3. **Apply snap in `onPointerMove`** for:
   - Single node drag (`dragTarget.kind === 'node'`)
   - Multi node drag (`dragTarget.kind === 'nodes'`)
   - Group drag (`dragTarget.kind === 'group'`)
   - Node resize (`node-resize`) ŌĆö snap resulting size and position
   - Group resize (`group-resize`) ŌĆö same
   - **Skip:** connection-endpoint, pan, marquee (D-06)

4. **Toolbar:** Add toggle button after Pan tool with `grid_on` / `grid_off` Material icon; `aria-pressed`; default active.

5. **Keyboard:** Track `shiftKey` in existing keydown/keyup handlers (Space already tracked for pan).

## Export Safety (must not regress)

- Golden SVG suite must pass unchanged (D-09, D-10).
- Text export tests must pass unchanged (D-12).
- Optional fixture with `diagram.grid` in YAML should render identical SVG to equivalent diagram without grid block.
